# Technical overview

This section is for **developers and coding agents** who need to build on Agentron: extend the runtime, integrate with the API, or understand functions and modules in the codebase.

## Source of truth

- **Code**: [GitHub repository](https://github.com/cap-jmk-real/agentron). Types and runtime live in `packages/core` and `packages/runtime`.
- **Contracts**: Repo `contracts/` (e.g. `server-api.md`, `shared-types.md`) and `docs/` for design and integration notes.

When integrating with external APIs or protocols, check the repo and official docs rather than assuming request/response shapes.
