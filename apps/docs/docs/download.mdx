# Download Agentron Studio

Get the **desktop app** for Windows, macOS, or Linux. The download picker below detects your platform and lets you choose a different one if needed. Builds are produced automatically from GitHub Releases when we tag a new version (e.g. `v0.1.0`).

<DownloadLinks />

## Running the desktop app

After installing, start the **web UI** (e.g. `npm run dev:ui` from the project root or a pre-built deployment), then launch the desktop app. It will connect to the UI at the URL you configure or at `http://localhost:3000` by default. See the [intro](/docs/intro) page for install options and links.
